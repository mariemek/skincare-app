# Skincare Routine Tracker

A mobile skincare routine tracker built with **React Native (Expo)**.  
This app helps users organize their skincare products, schedule routines, and track daily progress to build consistent skincare habits.

---

## Features

- Manage a personal collection of skincare products
- Schedule products for **morning and evening routines**
- Track daily routine completion
- View progress with completion percentage
- Weekly routine planner
- Add product images
- Clean and modern mobile UI

---

## App Screens

### Home
Displays the selected day's skincare routine and shows progress for completed products.

### Products
Add, view, and manage skincare products in your collection.

### Schedule
Create and manage a weekly skincare routine by assigning products to specific days and times.

### Profile
View skincare statistics and helpful skincare tips.

---

## Tech Stack

- **React Native**
- **Expo**
- **SQLite** (local database)
- **React Navigation**
- **Expo Image Picker**
- **Expo Vector Icons**
